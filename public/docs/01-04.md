Chapter: キャラクター
Section: 装備
---
お金の単位は「金」とします。初期所持金は誰でも例外無く100金です。

装備一覧からスキルに応じた好きな武器と防具を選び、必要な数値を書き写します。必要筋力のような概念はありません。

#### 武器

###### 表: 武器と必要数値

| 数値        | 説明                                                           |
| --------- | ------------------------------------------------------------ |
| Dmg(ダメージ) | 攻撃力を表します。dはダイスの数です。  <br>武器の基本ダメージに、本人の筋力(ST)によるダメージを足した値です。 |
| Lv(レベル)   | 技能値を書き写します。命中判定の目標値になります。                                    |
| P-EV(受け)   | 「受け」を試みるとき、武器によって有利に補正される回避判定の修正値を表します。                      |

#### 防具

###### 表: 防具と必要数値

| 数値         | 説明                                      |
| ---------- | --------------------------------------- |
| DR(ダメージ抵抗) | 防御力を表します。                               |
| B-EV(止め)    | 「止め」を試みるとき、盾によって有利に補正される回避判定の修正値を表します。  |
| WT(重量)     | 「よけ」を試みるとき、重量によって不利に補正される回避判定の修正値を表します。 |

###### 表: 防具（盾）一覧

| 盾    | B-EV | 金額  | 技能  | 備考     |
| ---- | --- | --- | --- | ------ |
| 小盾   | 2   | 10金 | 剣術  | -      |
| 中盾 W | 3   | 15金 | 武術  | -      |
| 大盾 W | 4   | 20金 | 武術  | 命中判定-2 |

###### 表: 防具（服・鎧）一覧

| 服・鎧     | DR    | WT  | 金額   | 技能  | 備考   |
| ------- | ----- | --- | ---- | --- | ---- |
| 服       | 1 (0) | 0   | 20金  | -   | -    |
| 革服      | 1     | 0   | 40金  | -   | -    |
| 革鎧      | 2     | 1   | 80金  | -   | -    |
| チェインメイル | 3 (1) | 1   | 120金 | -   | -    |
| ブリガンディ  | 3     | 2   | 160金 | 武術  | DX-1 |
| ハーフプレイト | 4 (2) | 2   | 240金 | 武術  | DX-1 |
| プレイトメイル | 4     | 3   | 320金 | 武術  | DX-2 |

※繊維構造の服やチェインメイルの刺突攻撃に対するDRは()内の値を用います。

片手持ちの武器を選んだ場合は、盾も忘れずに購入しましょう。どの武器を選んだとしても、それぞれ状況によって個性を発揮しますが、「どれが強くて、どれが弱い」ということはありませんので、人物のイメージで決めてしまってください。

初期所持金の100金では、十分な装備が揃わないのが普通ですが、PC間で金銭の貸し借りは原則禁止です。
