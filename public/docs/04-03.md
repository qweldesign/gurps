---
Chapter: 法術
Section: ファンブル
---
人物が恐怖を感じる状況が発生したら、恐怖を克服できるかどうか、恐怖判定を行います。まずIN(知力)-2または「修養」判定を行い、これに失敗すると恐怖を抑えられません。判定の失敗度に3dを足して、感じた恐怖の度合いを判定します。

また、法術のファンブルが発生した場合も同じような処理をたどります。失敗度に3dを足し、どんな大失敗をしたか判定します。

| 出目   | 効果   | 内容                                                                                                                                                          |
| ---- | ---- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 10以下 | 朦朧/軽 | 精神的な朦朧状態に陥る。  <br>毎ターン開始時に、IN(知力)-2または「修養」による回復判定。                                                                                                         |
| 11   | 朦朧/中 | 精神的な朦朧状態に陥る。  <br>毎ターン開始時に、-1の修正でIN(知力)-2または「修養」による回復判定。                                                                                                   |
| 12   | 朦朧/重 | 精神的な朦朧状態に陥る。  <br>毎ターン開始時に、-2の修正でIN(知力)-2または「修養」による回復判定。                                                                                                   |
| 13   | 不運   | 12の症状に加え、なぜか「不運」を背負う。  <br>当面、何か不運な事が起きるとすれば、その人物に降りかかる。  <br>法術ファンブルの場合、術が思いもよらない人物にかかってしまう。  <br>このとき誰にかかるかはゲームが面白くなるようにGMが決める。                           |
| 14   | 失声   | 12の症状に加え、3d分間、喋ることができない。  <br>法術の詠唱も不能。                                                                                                                     |
| 15   | 思考不能 | 重度の精神的な朦朧状態に陥る。  <br>毎ターン開始時に、-4の修正でIN(知力)-2または「修養」による回復判定。                                                                                                |
| 16   | 混乱   | パニックを起こし、叫びながら走ったり、座り込んで泣いたりする。  <br>法術ファンブルの場合、思いもよらない上位の別の術が発動する。  <br>このとき何が発動するかはゲームが面白くなるようにGMが決める。  <br>直後に修正無しでWLによる回復判定を行い、成功すれば15へ。  <br>失敗すれば17へ。 |
| 17   | 大混乱  | 16の症状が持続する。  <br>再び-2の修正でWLによる回復判定を行い、成功すれば16へ。  <br>失敗すれば18へ。                                                                                              |
| 18以上 | 卒倒   | 17の症状が持続する。  <br>再び-4の修正でWLによる回復判定を行い、成功すれば17へ。  <br>失敗すれば卒倒し、昏睡状態に陥る。  <br>部位の「故障の期間」を参考に回復に要する時間を決める。  <br>13以上の出目を振ると、永久的にIN(知力)が1減じる。                   |
