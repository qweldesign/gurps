# GURPS.QWEL

## 概要

- React を使用した創作 TRPG シミュレーター / 個人の学習用途を兼ねた実験プロジェクト
- DEMO: [https://gurps.qwel.design/] (ドメインを取得したら変更の可能性あり)

---

## 開発経緯等

- 筆者はテーブルトークRPG (TRPG) をやったことは無いが, ルールブックを読むのが昔から好きだった
- GURPS の戦闘シーンは, 状況を簡単に数値化できない面白さがある
  - 例えば, ファンブルで武器を落としたり, 壊したりする → 格闘戦や落した武器の奪い合いが始まる
  - 魔法の効果がハチャメチャ → 死人を操ったり, 時間を巻き戻したり (ただし, これは筆者独自に考案)
- 魔法に関しては, 「サガ・シリーズ」の影響を多々受けていますが, 二次創作です

## 開発状況

- 2016頃 GURPS を簡略化したゲームアルゴリズムの構想開始, ドキュメント着手

- 2021.01 戦闘シーンの大部分の再現にまで至ったが、開発断念

### ベータ版 (現行)

- 2026.02 ゲームアルゴリズムの修正, ドキュメント原案作成
- 2026.03 ゲームシステム構成設計
- 2026.03.28 ドキュメントのレンダリング機能を実装
