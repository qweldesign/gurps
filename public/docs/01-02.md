#### CP

プレイヤーの分身となる人物(PC)の作成は、持ち点（CP:キャラクターポイント）を10点とし、それをST(筋力)、DX(敏捷力)、IN(知力)、HT(生命力)の4つの「能力値」、および技能に振り分けることから始めます。CPは1/2CP単位で消費することができます。

HP(耐久点)と技能にもCPを使うので、1〜3点ほど残しておいてください。

#### 能力値

能力値の初期値はどれも8ですが、値を大きくするほど必要な点数も等比数列的に増えていきますので、能力値の説明と、能力値と消費CPについては、下記表を参考にしてください。

###### 表: 能力値の説明

| 能力値     | 説明                  |
| ------- | ------------------- |
| ST(筋力)  | 力の強さ。戦闘での加ダメージに関係。  |
| DX(敏捷力) | 工夫や閃き。戦闘での回避率に関係    |
| IN(知力)  | 知識と感性。法術（魔法）の修得に関係。 |
| HT(生命力) | 根性。戦闘での被ダメージに関係。    |

###### 表: 能力値と消費CP

| 能力値 | 消費CP      |
| --- | --------- |
| 8   | 0CP       |
| 9   | 1/2CP     |
| 10  | 1CP       |
| 11  | 2CP       |
| 12  | 4CP       |
| 13  | 8CP       |
| 以降  | +8CPずつ増える |
